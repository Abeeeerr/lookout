# Lookout

> Spot the best apps & utilities for your Mac — a playful, interactive discovery
> experience wrapped in a colorful retro-Apple aesthetic.

**Live site:** https://abeeeerr.github.io/lookout/

Lookout reimagines the giant "awesome-mac" lists as a faux classic-Macintosh
desktop: browse categories in draggable Finder-style windows, open apps for a
"Get Info" view, and jump straight to their official sites.

## Tech

- React + TypeScript + Vite
- Framer Motion (animations) · Zustand (window manager)
- Fully static — deployed to GitHub Pages via GitHub Actions

## Develop

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # production build to dist/
```
